A la découverte de la vie multilingue des mots

Les mots ont plusieurs vies lesquelles se révèlent à travers de nombreux paramètres comme par exemple : le contexte, la langue, l'époque à laquelle ils sont employés etc. L'objectif de ce projet est d'étudier la vie linguistique d'un mot cible dans plusieurs langues différentes, d'analyser et de comparer leurs usages.

Méthodologie

Le projet étape par étape

  • Le choix d’un mot à explorer
  • La recherche d’URLs pour étudier le multilinguisme du mot choisi
  • La formation d’un corpus via la récupération des données textuelles et autres informations récupérées depuis les URLs sélectionnés
  • La construction d’un nuage de mot
  • L’étude des contextes gravitant autour du mot choisi (concordancier)

Script de récupération de pages internet

Construction d'un script, implémenté au fur et à mesure, pour répondre à plusieurs objectifs :

  1. Lire une liste d’URLs donnée
  2. Vérifier la validité de ces URLs
  3. Vérifier la réponse du serveur des URLs valide pour savoir si la page est accessible
  4. Récupérer l’encodage des pages accessible
  5. Extraire le contenu textuel des pages (les convertir en UTF-8 si nécessaire)
  6. Construire un tableaux qui rassemblent diverses informations tel que :
    • L’URL lui même
    • Le code retour de HTTP
    • L’encodage initial de la page
    • La page HTML brut (aspiration)
    • Le dump textuel
    • Le nombre d’occurence du mot cible dans la page
    • Le concordancier
    • Le contexte (1 par URL)
